# CODEX_V4_MP

Учебный проект по тестовому заданию на 1С:Предприятие 8.3.

В репозитории хранится выгрузка конфигурации в файлы, с которой удобно работать через Git:
- [Configuration.xml](./Configuration.xml)
- [Catalogs](./Catalogs)
- [Documents](./Documents)
- [Languages](./Languages)

## Что сейчас есть

- справочник `Номенклатура` с реквизитом `Артикул`
- подчиненный справочник `Характеристики`
- в `Характеристиках` используются поля `Цвет` и `Размер`
- `Наименование` характеристики формируется автоматически в модуле объекта
- документ `ДвижениеТоваров` с табличной частью `Товары`

## Что пока не завершено

- форма документа и отбор характеристик по номенклатуре
- регистр накопления и движения документа
- контроль остатков при проведении
- отчет на СКД
- HTTP-сервис

## Как использовать

1. Открыть информационную базу в Конфигураторе.
2. Загрузить конфигурацию из файлов этого репозитория.
3. Обновить конфигурацию базы данных.
4. Проверять изменения в режиме `1С:Предприятие`.
