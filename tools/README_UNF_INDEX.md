# Быстрый доступ к примерам из УНФ

Чтобы не использовать тяжелую папку `УНФ` вручную каждый раз, в проект добавлен локальный индекс.

## Как обновить индекс

```powershell
python .\tools\index_unf.py
```

## Что создается

- `tools/unf_index.json` — машинный индекс по объектам
- `tools/unf_index.md` — краткая сводка

## Что удобно искать

- подчиненные справочники
- наличие стандартного реквизита `Owner`
- формы объекта
- модули объекта и менеджера
- примеры документов с `Характеристикой`

## Как я буду это использовать дальше

1. Ищу подходящий объект в `tools/unf_index.json`.
2. Открываю только нужный XML или модуль из `УНФ`.
3. Забираю точечный шаблон, не читая всю выгрузку.
